## Choosing a theme

The **Page Editor** includes a theme selector — a default theme plus genre-based
themes. Pick the one that fits your band's look; it applies across all your blocks.

## Custom CSS

For finer control, add **custom CSS** in the editor. A few notes:

- There's a size limit (around 50KB) — plenty for styling tweaks.
- CSS is **sanitized** for safety, so some properties may be stripped.
- Start from the theme and layer small overrides rather than restyling everything.

## Tips

- Preview frequently — use the page's preview link to check your changes live.
- If something looks off after a CSS change, remove the rule and re-add it
  piece by piece to find the culprit.
