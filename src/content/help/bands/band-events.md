## Band events

Owners and admins can create events for the band from the band's **Events** page —
a list of the band's shows with their status (draft, published, or cancelled).

## Creating an event

1. Open the band's **Events** page and start a new event.
2. Fill in the **title, description, date**, and **start/end times**.
3. Add **doors open** time, **location**, **tags**, and an external **ticket
   link** if you sell tickets elsewhere.
4. Save it as a draft.

## Managing an event

Open an event to edit its details. Admins can **publish** it (make it visible),
**unpublish**, or **cancel** it. Published events show up for the community.
