## Editing the profile

Owners and admins edit the band's public listing from the band's **Edit** page. It
works much like your personal profile.

## What you can set

- **Name, tagline & avatar** — The band's identity on its directory card.
- **Bio** — A rich-text description.
- **Genres** — Freeform tags (with suggestions) used in directory filters.
- **Links** — SoundCloud, YouTube, Spotify, and more; some support embeds.
- **Directory contact info** — A public email, phone, or social handle.
- **Visibility** — Public or members-only.
- **Looking for members** — Flag the band as open to new members.

Save when you're done. For a richer public site, see
[Premium band pages](/member/help/premium-band-pages).
