## Notification preferences

CorvMC notifies you about things like reservation confirmations, payment
reminders, and band invitations. You control how you hear about each type.

## Adjusting them

1. Open **Account** from the sidebar.
2. Find the **Notifications** section. Each notification type has its own row.
3. For each type, toggle whether you want it delivered by **email**, **in-app**,
   or both.
4. Changes save as you make them.

Turning everything off for a type means you won't be reminded about it — keep
reservation and payment notifications on if you want booking reminders.
