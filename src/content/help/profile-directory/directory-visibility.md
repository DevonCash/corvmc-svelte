## Visibility levels

Your **Profile** page has a visibility setting that controls who can find your
listing in the directory:

- **Public** — Anyone, including visitors who aren't signed in, can see your
  profile.
- **Members only** — Only signed-in CorvMC members can see it.
- **Hidden** — Your profile doesn't appear in the directory at all.

## Choosing a level

- Pick **Public** if you want to be discoverable by anyone (e.g. bands or venues
  scouting players).
- Pick **Members only** to share with the community but stay off the public web.
- Pick **Hidden** if you'd rather not be listed right now. You can switch back any
  time.

If your profile is hidden, some pages will offer a shortcut to make it visible
again so you don't get stranded off the directory.
