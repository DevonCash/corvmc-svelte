Type · feature settings

Lexend variants

Lexend ships a generous set of OpenType axes that let one font feel like several. Toggling font-feature-settings swaps in alternate letterforms without changing the family. The CMC system defaults to cv05 + cv11 — single-story g and a — which give Lexend a softer, mid-20th-century humanist character. The rest of these are opt-in per-element when you want more personality.

Usage. Body and headings inherit var(--font-features-body) and var(--font-features-display) from colors_and_type.css — you usually don't need to think about it. Reach for 'tnum' on data tables and stat cards, 'zero' on monospaced codes, and 'frac' / 'ordn' on prose that contains them. Avoid stacking everything — the family stays cleaner with restraint.
