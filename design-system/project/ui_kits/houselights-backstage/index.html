Same building. Different time of day.

CMC ships two themes — Houselights and Backstage — and every piece of the design system renders in both. Same chrome, same functional UI, same decorative objects. The room changes; the kit doesn't. A booking table can be either theme. A poster wall can be either theme.

01 · Three principles

The system has three load-bearing layers.

Every piece belongs to one of three layers — Chrome (UI shell — buttons, surfaces, badges), App (functional dashboard components — tables, forms, KPIs), or Decorative (skeumorphic program objects). Each layer ships in both themes.

01

The room changes. The DNA stays.

Both rooms run on the same speaker mark, the same six brand hues, the same Lexend type, the same tri-stripe. The difference is the lighting on top.

02

Three layers. Both rooms.

Chrome (shell), App (functional dashboard UI), Decorative (skeumorphic objects). Every layer renders in both themes. Pick the layer for the job, not the theme.

03

The tri-stripe is the constant.

Teal / goldenrod / red-orange, exactly the same recipe in both rooms. Strongest brand recall device, never tuned.

03 · Materials

Eight surfaces. Both themes use the same materials.

Real-world materials show up across the decorative layer. Gaffer tape is gaffer tape whether the lights are on or off. Spike tape is spike tape. Parchment, paper, and glow tape behave the same way.

04 · Hand-lettering

Two handwriting roles. Two different tools.

Lexend still handles every UI string. Handwriting only appears where a real backstage object would actually be hand-labelled — a paint pen on a strip of tape, a Sharpie on a setlist. Same two roles in both rooms.

Fragile
Chalkpen · Permanent Marker
Paint-pen on gaffer tape. All-caps, single weight, slight rotation. Used on tape labels, hazard tags, primary CTAs in tape form, marker annotations (ENCORE, SOLD OUT, DEMO). Never in body copy or product UI.

Sharpie · Caveat
Ballpoint or Sharpie on paper. Mixed case, leans right, lighter weight. Used on setlist items, taped-up reminders, hand-written captions on cassette labels and ticket stubs.

07 · Decorative — skeumorphic program objects

09 · Theme picking

How a user lands on each theme.

Theme isn't a use-case split — both themes ship every layer of the system. What differs is just the user's lighting preference.

Default · light theme

Houselights

  • Default for unsigned-in users and users with light OS preference
  • Recommended canvas for print, email, posters, slides
  • Renders every layer: chrome, functional, decorative
  • Activated by data-theme="light" or no attribute
Opt-in · dark theme

Backstage

  • Users with prefers-color-scheme: dark
  • Users who flip the explicit theme toggle
  • Often paired with long-session work — but not required
  • Activated by data-theme="dark"
