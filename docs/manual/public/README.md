# Public Site Manual

How-to articles for the **public** (signed-out) CorvMC site. These live here as
markdown rather than in the in-app Help/KB, because `/member/help` is behind member
auth — there's no public help route yet (see the
[manual manifest follow-ups](../README.md#follow-ups-not-in-the-first-pass)).

Each file carries the same frontmatter shape as the in-app articles, so if a public
help route is added later, they can be moved into `src/content/help/` (category
`public`) and synced with little change.
